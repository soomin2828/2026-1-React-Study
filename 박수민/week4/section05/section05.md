**5.3 JSX**

자바스크립트에서는 함수가 html을 return하도록 만들 수 없지만 리액트에서는 jsx문법을 사용하여 가능함

jsx: 확장된 자바스크립트의 문법

jsx 문법 내에서 자바스크립트의 값을 html로 렌더링하고 싶다면 중괄호 안에 작성하면 됨

중괄호 안에는 숫자나 문자열로 표현된다면 뭐든 가능 삼항 연산자도 가능





JXS **주의 사항**

1. 중괄호 내부에는 자바스크립트 표현식만 넣을 수 있음

if 조건문 XX

for 반복문 XX

한 줄로 값으로 평가되어야함



2\. 숫자, 문자열, 배열 값만 정상적으로 렌더링 됨

불리안, null, 객체값 렌더링 안됨



3\. 모든 태그는 닫혀있어야 함



4\. 최상위 태그는 반드시 하나여야만 함

빈태그로 묶어주면 흩뿌려져 있음





스타일 적용 방법

1. 직접 스타일 속성 설정 {{로 스타일 지정, -없이 써야함
연결되는 단어의 첫 시작은 대문자로 표기 (카멜 표기법)
2. 별도의 css파일로 지정

&#x09;css 파일을 불러옴(경로만 불러와도 됨)

&#x09;className으로 표현

**5.6 State - 상태 관리하기**

state 현재 가지고 있는 형태나 모양을 정의하며 변화할 수 있는 동적인 값

state 컴포넌트의 형태 상태를 보관하는 변수로 state의 값에 따라 렌더링 되는 UI가 결정됨

하나의 컴포넌트에 여러 개의 state가능

리렌더링

컴포넌트 state 값이 바뀌면 컨포넌트가 다시 return함 이때 변경된 state 값도 다시 반영됨



**5.7 State / Props**

자식 컨퍼런트는 자신이 갖는 state가 변경되지 않아도 부모 컨퍼런트로부터 받는 props값이 바뀌게 되면 리렌더링이 발생



리렌더링이 되는 상황 3가지 

1. 자신이 관리하는 state값이 변경되었을 때
2. 자신이 제공받는 props값이 변경되었을 때
3. 부모 컨퍼런트가 리렌더링 될 때



관련이 없다면 서로 다른 컨퍼런트로 분리

**5.10 useRef로 컨포넌트의 변수 생성하기**

useRef: 새로운 Refernece 객체를 생성하는 기능

const refObject = useRef()

refObject: 컨퍼넌트 내부의 변수로 일반적인 값 저장 가능



useRef: Reference 객체 생성

&#x20;**5.11 React Hooks**

&#x20;react hooks: 클래스 컴포넌트의 기능을 함수 컴포넌트에서도 이용할 수 있도록 함

useState, useRef도 모두 react hooks임

각각의 메서드를 hook이라고 부름

hook들은 함수 컨퍼넌트 내부에서만 호출 가능

조건문, 반복문 내부에서는 호출 불가

use(접두사)를 이용하여 나만의 hook을 만들 수 있음



hook 관련 팁 3가지

1. 함수 컴포넌트, 커스텀 훅 내부에만 호출 가능
2. 조건부로 호출될 수 없음

서로 다른 훅들의 호출 순서가 엉망이 되기 때문

3\.	커스텀 훅을 만들 수 있음

일반 자바스크립트에서 훅을 호출하려면 오류

반복 로직 분리 가능
